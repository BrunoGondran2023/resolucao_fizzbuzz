# FizzBuzz

Este é um projeto simples que resolve o desafio FizzBuzz usando Python.

## Descrição

O desafio FizzBuzz é um exercício clássico de programação que imprime os números de 1 a n, substituindo múltiplos de 3 por "Fizz", múltiplos de 5 por "Buzz" e múltiplos de ambos por "FizzBuzz".
